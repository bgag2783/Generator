Follow these steps to deploy your website using GitHub Pages:

  1. Create a GitHub Repository
    Go to GitHub and create a new repository.
  2. Upload Your Site Files
    - Extract the downloaded ZIP file.
    - Drag & drop all files into your new GitHub repository.
  3. Enable GitHub Pages
    - Go to your repository Settings.
    - Scroll down to the GitHub Pages section.
    - Select the branch (usually main) and save.
  4. Access Your Live Site
    GitHub will generate a link like: https://your-username.github.io/repo-name/.
